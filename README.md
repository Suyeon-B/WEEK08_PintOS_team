# 🔥 PintOS-Kaist 프로젝트

- 프로젝트 기간 & 인원 : 2022.05 - 2022.06 (6주) / 3명
- PintOS의 코드를 직접 수정해가며 진행하는 프로젝트입니다.
- PintOS는 2004년 스탠포드에서 만들어진 교육용 운영체제입니다.
- 이를 기반으로 KAIST 권영진 교수님 주도 하에 만들어진 `KAIST PintOS`로 진행됩니다.
